ShooterGame

Download

  • ShooterGame customized version v0.1.0 (diff zip file)

Install

  1. Download the "ShooterGame" project from the "Epic Games Launcher" and deploy it to your PC.
  2. Download one of the "ShooterGame Customized Version" differential zip files listed above.
  3. Unzip the file and overwrite the "ShooterGame" project folder with the extracted files.
  4. Build the "ShooterGame" project.
  5. If the build fails, delete the "ShooterGame.automation" project from "Solution Explorer" and try building again.
  6. If the build is successful, right-click the "ShooterGame" project on "Solution Explorer" to open the menu and select "Set as Startup Project."
  7. To open PIE (Play In Editor), run "Local Windows Debugger" or "Start Without Debugging".
